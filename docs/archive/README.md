# Documentation Archive

**Purpose:** Superseded, obsolete, or duplicate documentation is stored here. These files are kept for historical context and are not the current source of truth.

**When to archive:** A doc is moved here when it has been superseded by a canonical doc, is a one-time deliverable/audit with no ongoing use, or is an old phase summary replaced by a consolidated document.

**Archive note:** Each archived file may have a short note at the top indicating why it was archived and what supersedes it. Some internal links in archived docs still point to `docs/...`; if the target was moved here, it will be in the same archive folder.
